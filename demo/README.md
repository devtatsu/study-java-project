## 準備

### vsCode

- Extension Pack for Java
- Spring Boot Extension Pack
- Lombok Annotations Support for VS Code

## 実行

- サービス起動

`demo`直下で、以下のコマンドを実行。

`./gradlew bootRun`

- 確認

サービス起動後、以下にアクセスして、確認。

`http://localhost:8080/greeting`
